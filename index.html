📊 Índice de Shock

🧮 ¿Cómo se calcula?

Fórmula: Índice de Shock = Frecuencia Cardíaca ÷ Presión Arterial Sistólica

Ejemplo: FC = 110 lpm, PAS = 90 mmHg → Índice = 110 ÷ 90 = 1.22

📌 Interpretación Clínica

Índice Significado
< 0.7 Normal
0.7 – 0.9 Compensación hemodinámica
0.9 – 1.2 Shock incipiente
> 1.2 Shock significativo
